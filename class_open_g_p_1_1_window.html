Constructor & Destructor Documentation

OpenGP::Window::Window ( std::function< void(Window &)>  display_callback)

Create and show a new window.

display_callback will be executed when draw() is called, and should perform any rendering needed to update the content of the window.

GLFWwindow *  parent_context

Create and show a new window that shares objects with a parent context.

display_callback will be executed when draw() is called, and should perform any rendering needed to update the content of the window. Any transferable OpenGL objects in parent_context will be inherited
